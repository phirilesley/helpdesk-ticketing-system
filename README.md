## 📌 Overview

An enterprise-grade help desk system with real-time chat, workflow automation, and SLA tracking.

---

## 🎯 Purpose

To demonstrate the ability to build:

* Workflow-driven systems
* Enterprise ticket management platforms

* SLA Tracking

* Reports & Analytics
